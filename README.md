# CodeAlpha_Image-Gallery

A modern and responsive image gallery developed using HTML, CSS, and JavaScript. The project includes a clean grid layout, hover effects, smooth transitions, lightbox image preview, next/previous navigation, and category-based image filtering.

## Features
- Responsive gallery layout
- Lightbox image preview
- Next and previous navigation
- Hover animations and smooth transitions
- Image category filtering
- Mobile-friendly design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in a web browser
